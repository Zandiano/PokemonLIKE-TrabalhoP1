# PIKOMON

## Descrição-
Um rpg de turno baseado nos primeiros jogos de final fantasy e pokemon, com o uso de um inimigo inteligente que segue o jogador e sabe a probabilidade de perder a batalha, o jogo possui um menu, um mapa aberto, um sistema de batalha e uma tela de game over com scores

## Bibliotecas-
gconio.h 

stdio.h 

stdlib.h 

math.h 

time.h
